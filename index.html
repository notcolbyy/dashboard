Setup Guide

Step-by-step instructions to connect everything.

  1. Go to console.cloud.google.com
  2. Create a project, enable Calendar, Classroom & Gmail APIs
  3. Create OAuth 2.0 credentials → Web Application
  4. Add your GitHub Pages URL as an authorized origin
  5. Paste your Client ID above and click Save & Connect
  6. Get a free Finnhub key at finnhub.io
  7. Get a free OpenRouter key at openrouter.ai/settings/keys
